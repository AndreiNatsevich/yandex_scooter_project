1. Вывести список логинов курьеров с количеством их заказов в статусе «В доставке» (inDelivery = true)

SELECT c.login, COUNT(o.id) AS "ordersCount"
FROM "Couriers" AS c
INNER JOIN "Orders" AS o ON c.id = o."courierId"
WHERE o."inDelivery" = true
GROUP BY c.login;


2. Вывести все трекеры заказов и их статусы по правилу: finished == true → 2; cancelled == true → -1; inDelivery == true → 1; иначе 0

SELECT track,
       CASE
           WHEN finished = true THEN 2
           WHEN cancelled = true THEN -1
           WHEN "inDelivery" = true THEN 1
           ELSE 0
       END AS status
FROM "Orders";
